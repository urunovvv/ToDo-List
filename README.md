# :memo: Console To-Do List

Простая консольная программа для управления задачами на Java. Позволяет добавлять, удалять, отмечать выполненные задачи, фильтровать по приоритету и многое другое.

## :hammer_and_wrench: Функционал

- ✅ Добавление задач с заголовком, описанием и приоритетом
- ✅ Просмотр всех задач
- ✅ Отметка задач как выполненных
- ✅ Удаление задач
- ✅ Редактирование заголовка задачи
- ✅ Просмотр только невыполненных задач
- ✅ Фильтрация задач по приоритету (HIGH / MIDDLE / LOW)
- ✅ Очистка консоли

##  🏗️ Архитектура

Проект построен с соблюдением принципа разделения ответственности:

| Класс | Ответственность |
|-------|-----------------|
| `Main` | Точка входа, сборка зависимостей |
| `Task` | Модель данных задачи |
| `TaskRepository` | Хранение данных (CRUD операции) |
| `TaskService` | Бизнес-логика |
| `ConsoleUI` | Пользовательский интерфейс (консоль) |
